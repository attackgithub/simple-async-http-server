# Простой сервер на C# бес ипользования WCF и ASP.NET

Сервер написан на языке C# и осуществляет асинхронную обработку запросов, умеет работать с БД xml и SQLite.
Написан с использованием методологии TDD. Для логирования применяется пакет NLog.

Стоит отметить что сервер написан как демо приложение, поэтому в нем не хватает некоторых очевидных функци и оптимизаций. 

## Прокси сервер

В состав сервера включен простой прокси сервер.
Для его работы отправьте запрос GET на <адрес_сервера>/proxy/?url=<адрес_сайта> 
где <адрес_сервера> по умолчанию 127.0.0.1:8888
<адрес_сайта> адрес в формате http://www.ya.ru

## Гостевая книга

Чтобы получить все записи из гостевой книги отправьте GET запрос на адрес <адрес_сервера>/guestbook/
Чтобы добавить запись в гостевую книгу отправьте POST запрос на адрес <адрес_сервера>/guestbook/?user=<username>&?message=<message>

# Установка и настройка
## Сборка

Проект представляет собой консольное приложение. Для начала работы, требуется
собрать проект VisualStudio 2010 AnyWayAnyDayDemo.sln из прилагаемой папки. 
Для успешной сборки убедитесь, что установлен пакет NLog и существует референс на библиотеку для работы
с БД SQLite.

После успешной сборки проекта, запустите его с помощью клавиши F5 или непосредственно из папки bin.

## Настройки
Базовые настройки сервера находятся в файлt server_config.xml. 
Дополнительно можно поиграться с параметрами файла ServerHelpers.cs
